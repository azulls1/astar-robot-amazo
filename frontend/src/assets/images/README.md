# `assets/images/`

Imágenes raster (PNG/JPG/WebP) e ilustraciones del producto.

## Qué va aquí

- Capturas del solver corriendo (estado inicial, mid-run, resultado).
- Diagramas y figuras del enunciado del problema.
- Ilustraciones decorativas para hero sections.
- Fondos/texturas si aplica.

## Disponibles para copiar al frontend

Hay imágenes ya generadas en otras carpetas del proyecto que podrían
incorporarse aquí si las quieres mostrar en la UI:

- `Requerimientos de la actividad/imagenes/image1.png` — diagrama del estado inicial
- `Requerimientos de la actividad/imagenes/image2.png` — diagrama del estado objetivo
- `astar_robot_amazon/memoria/capturas/*.png` — capturas para la memoria DOCX

## Acceso

`<img src="/assets/images/nombre.png" alt="..." />`
