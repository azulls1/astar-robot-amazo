Logos del producto, equipo, universidad y entidades relacionadas.

## Logos incluidos

| Archivo | Uso |
|---|---|
| `astar-logo.svg` | Logo del producto · cuadrado · paleta Forest |
| `astar-logomark.svg` | Solo el símbolo "A*" sin fondo · para favicon o usos compactos |

## Convenciones

- **Solo SVG** — escalable y editable.
- Variantes claro/oscuro como `logo.svg` y `logo-on-dark.svg`.
- Tamaño nativo del `viewBox` cuadrado (recomendado 64×64 o 128×128).
